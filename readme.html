📚 Flask Book Project

პროექტის აღწერა

ეს არის Flask-ის გამოყენებით შექმნილი წიგნების ვებ-აპლიკაცია. მომხმარებელს შეუძლია რეგისტრაცია, ავტორიზაცია, წიგნების დათვალიერება, მათი შეფასება და პროფილის მართვა.

ფუნქციები

  • მომხმარებლის რეგისტრაცია
  • ავტორიზაცია სისტემაში
  • წიგნების დამატება
  • წიგნების დეტალური ინფორმაციის ნახვა
  • შეფასებების დამატება და რედაქტირება
  • პოპულარული წიგნების გვერდი
  • მომხმარებლის პროფილი
  • „ჩვენ შესახებ“ გვერდი

გამოყენებული ტექნოლოგიები

  • Python
  • Flask
  • Flask-WTF
  • SQLAlchemy
  • HTML
  • CSS
  • SQLite

პროექტის სტრუქტურა

  • app.py – აპლიკაციის გაშვება
  • routes.py – მარშრუტები
  • models.py – მონაცემთა მოდელები
  • forms.py – ფორმები
  • templates/ – HTML შაბლონები
  • static/ – სტილები და სურათები
